# agent-skills

这个仓库当前只维护 `codex-starter`。

## 安装

在目标项目根目录执行：

```bash
bash /path/to/agent-skills/codex-starter/install.sh
```

安装后的运行时权威以目标仓库中的以下文件为准：

- `AGENTS.md`
- `.agents/skills/*/SKILL.md`
- `.codex/routing-policy.md`
