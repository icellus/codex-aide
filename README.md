# agent-skills

这个仓库包含两个 starter：

- `codex-starter`
- `claude-starter`

## 安装说明

### `codex-starter`

在目标项目根目录执行：

```bash
bash /path/to/agent-skills/codex-starter/install.sh
```

### `claude-starter`

在目标项目根目录执行手动复制：

```bash
cp -a /path/to/agent-skills/claude-starter/CLAUDE.md .
cp -a /path/to/agent-skills/claude-starter/.claude .
```
