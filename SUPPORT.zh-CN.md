# 支持说明

[English](SUPPORT.md)

如有歧义，以英文主文件为准。

## 适合提 issue 的内容

- `codex-aide` 的实际缺陷、行为回归、文档错误
- 现有规则、脚本、模板之间的明显不一致
- 对外发布后可稳定复现的安装后行为问题
- 明确的功能建议或治理规则改进建议

## 不适合提 issue 的内容

- 与本仓库无关的业务代码问题
- 目标仓库自己的环境、权限、CI 或部署故障
- 缺少最基本上下文、无法定位范围的泛化抱怨
- 安全漏洞或凭证泄露

安全相关问题请改走 [SECURITY.zh-CN.md](SECURITY.zh-CN.md)。

## 提问前建议准备

- 使用的文件路径、命令或入口
- 期望行为与实际行为
- 最小复现步骤
- 相关日志、报错或校验输出
- 影响范围：仅本仓库、仅安装后的目标仓库、还是两者都有

## 推荐入口

- 缺陷反馈：GitHub Issues 中的 Bug 模板
- 功能建议：GitHub Issues 中的 Feature 模板
- 已有方案准备提交：直接发 Pull Request，并按模板填写验证信息

## 响应边界

- 本仓库按维护者可用时间持续维护，不承诺固定 SLA
- 会优先处理可复现、影响范围明确、证据充分的问题
- 对缺少上下文的问题，维护者可能先要求补充信息，再决定是否继续处理
