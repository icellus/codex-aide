# 行为准则

[English](../CODE_OF_CONDUCT.md)

如有歧义，以英文主文件为准。

本仓库希望保持直接、专业、可协作的讨论环境。

## 我们期望的行为

- 基于事实、证据和可复现信息讨论问题
- 尊重不同经验背景、技术路线和表达方式
- 对代码、文档和流程提出明确、可执行的反馈
- 在发现误解或错误时，优先澄清问题而不是升级冲突
- 对新贡献者保持基本耐心，尤其是在首次提 issue 或 PR 时

## 不被接受的行为

- 人身攻击、羞辱、歧视、骚扰或威胁
- 明知会引发冲突，仍持续进行挑衅、刷屏或恶意抬杠
- 公开泄露他人隐私、凭证、内部链接或其他敏感信息
- 在 issue、PR、评论中发布与仓库无关的广告、钓鱼或垃圾内容
- 故意歪曲事实、伪造验证结果或反复忽略项目维护规则

## 适用范围

本准则适用于本仓库中的 issue、pull request、代码评审、提交说明，以及与本仓库协作直接相关的公开交流场景。

## 执行原则

- 维护者可以删除不合适的评论、issue、PR 或提交关联讨论
- 对重复或严重违规行为，维护者可以限制互动、关闭讨论或拒绝贡献
- 处理时以保护协作环境、减少持续干扰为优先

## 举报与处理

- 如遇行为问题，请联系仓库维护者并附上必要上下文
- 若暂时只能通过 GitHub 公开入口联系，请不要公开敏感细节，只说明需要私下处理
- 安全漏洞、凭证泄露或其他敏感安全问题，不要按本文件公开反馈，请改走 [SECURITY.zh-CN.md](SECURITY.zh-CN.md)

## 补充说明

本准则不是法律文本，也不试图穷举所有场景。
对未明确列出的行为，维护者会按“是否破坏协作秩序、是否增加他人不必要成本、是否影响项目可持续维护”来判断。
